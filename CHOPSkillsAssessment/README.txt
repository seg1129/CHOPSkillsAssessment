This repository contains the entire project for the skills assessment. to download perform the following steps. 
 - download project 
 - in command line create a new virtual environment. $ virtualenv ENV 
 	- more info and virtualenv can be found here: https://virtualenv.pypa.io/en/stable/userguide/#usage
 - move project into the virtual environment file path
 - start up virtualenv. source bin/activate
 - make sure you are in 'CHOPSkillsAssessment' file and start up django project. python manage.py runserver
 - in a browser, navigate to the server address.
